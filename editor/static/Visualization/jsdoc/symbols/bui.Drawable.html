Class bui.Drawable


Extends bui.Observable.
The base class for every drawable item. As a general rule, the constructors of drawables should never be called directly but through the bui.Graph#add function.
Defined in: biographer-ui.js.

{bui.Drawable|Boolean} select(select)
Use this function to select the drawable. Selection is normally done by clicking on a drawable. Think of a file manager which provides functionality to, for example, select multiple files and apply actions to all selected files. If you omit the parameter the current selection status will be returned.
