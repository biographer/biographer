Class bui.Graph


Extends bui.Observable.
This class controls the whole graph and is responsible for the management of nodes and edges, i.e. drawables.
Defined in: biographer-ui.js.

{Boolean|bui.Graph} highPerformance(highPerformance)
A graph supports a high and low performance mode. This has implications on the way dragging and resizing is realized. When in high performance mode the SVG will be changed while dragging or resizing the node. In low performance mode this will only be done at the end of the dragging or resizing.
Parameters:
{Boolean} highPerformance Optional
Set the performance for this graph to high (true) or low (false). Omit to retrieve current performance setting.
Returns:
{Boolean|bui.Graph} If you pass a boolean to this function it will set the new value and return the instance of the object on which you called the function (fluent interface). If you don't pass a parameter the current setting will be removed.
